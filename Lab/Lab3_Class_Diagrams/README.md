ATM System Class Diagram

This diagram maps the structural relationships within a banking system.

    A Bank is associated with multiple ATMs.

    A Bank is composed of zero or more Accounts.

    Each Account is composed of zero or more Transactions.

    A Withdrawal represents a specific subclass of a Transaction.

Medical Clinic System Class Diagram

This diagram outlines the entities in a medical setting.

    Both Patient and Doctor inherit attributes from a common Person class.

    A Patient is linked to zero or more Medical Records.

    A Doctor interacts with multiple Medical Records.

    A Doctor operates within a Clinic.

Car Insurance Class Diagram

This diagram defines the core data structures for managing auto insurance.

    An Insurance Company issues multiple Policies.

    A Customer can hold multiple Policies.

    A Customer can own multiple Vehicles.

    A single Policy can be tied to multiple Claims.
